🎯 ¿Qué hace este agente?

📚 Búsqueda automática: Cada lunes a las 10:00h (hora Canarias), el agente consulta la base de datos Europe PMC (45M+ papers) con 27 keywords especializadas en arritmias y electrofisiología.
📊 Generación de reporte: Produce un resumen en formato Markdown + JSON con los papers de los últimos 7 días, ordenados por área clínica.
📱 Notificación instantánea: Envía un resumen ejecutivo al móvil vía Telegram Bot, con títulos, autores y DOI directo.
  • Mantenerme actualizado sin tiempo extra de búsqueda manual
  • Cobertura de 5 áreas críticas de arritmias y electrofisiología
  • Acceso inmediato a guías ESC/AHA/EHRA en el momento de publicación
  • Seguimiento de investigación emergente en IA + ECG y digital health
  • Coste €0/mes — 100% gratuito y automático

🔄 Flujo Semanal Automatizado

1
⏰ Lunes 10:00h — GitHub Actions activa el agente

El scheduler cron (0 9 * * 1 UTC = 10:00h Canarias) lanza automáticamente el script Python en los servidores de GitHub.

2
🔬 Búsqueda en Europe PMC

Se consultan 27 keywords en la API de Europe PMC, filtrando papers publicados en los últimos 7 días. La API devuelve título, autores, revista, fecha, DOI y abstract.

3
📊 Generación de reporte

El agente clasifica los papers por área clínica, elimina duplicados por DOI y genera dos archivos en /reportes: un Markdown legible y un JSON estructurado.

4
📱 Notificación Telegram

El bot envía al móvil un resumen con los papers más relevantes: título, primer autor, revista y enlace directo al paper. Disponible en segundos.

5
📂 Commit automático a GitHub

El reporte se guarda en el repositorio con fecha en el nombre de archivo (reporte_YYYY-MM-DD.md), creando un historial permanente y accesible.
